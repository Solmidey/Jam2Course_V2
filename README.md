# Run and deploy your Jam2Course app

This repository contains everything you need to run the Jam2Course application locally.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. (Optional) Create a [.env.local](.env.local) file to store any environment variables your deployment requires (e.g., `API_KEY`).
3. Run the app:
   `npm run dev`
